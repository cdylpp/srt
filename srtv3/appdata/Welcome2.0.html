Datasets are the primary object in any data analysis endeavor.Therefore, within the SRT framework, datasets are at the core.

Start: Import your own datasets

  1. Click the icon button or under File select Import Data.
  2. Use the file dialog to select a CSV file
  3. After import a new Tab will populate within the main screen.
  4. Navigate to this new Tab by clicking on the tab.

This new tab will present a DataDashboard, a view pane with several filter and sort options to view your dataset. Let's take a closer look at the widgets within the DataDashboard.

Descriptive Statistics üîç

The left column houses some descriptive widgets so you can quickly view each feature and gather a snapshot of what the data represents.

Underneath the Columns Header, click on the dataset name to view a drop-down list of all the attributes within the dataset.

The left column, DataTypes, describes the variable datatype within the analytics model engine. We plan to allow for the complex management of handling all data types. Currently, we can safely handle integers (int), floating-point numbers (float), and text (str or object).

To see some visualizations of an individual variable, click on the attribute name underneath the drop-down menu. A plot should be shown in the left column of the DataDashboard along with some summary statistics.

Pie Chart ü•ß

A pie chart is a circular statistical graphic that is divided into slices to illustrate numerical proportion. Pie charts are used to show proportions of a whole, and the arc length of each slice is proportional to the quantity it represents.

Report üìÑ

Clicking the Reports icon with an imported dataset will bring up the predicitive analysis screen. This view houses some unqiue features within SRT, democratizing machine learning for retention analysis. Adminstrative users can select a Target Variable, specify which machine learning model to use based on variable parameter, and define the test-train split size. Once these parameters are specified the user can run the model to get an output of the results.

Once ran, the view will populate with a Right Dock Widget showing each model and the accuracy, run time, and confusion matrix. When the user navigates to each model, the confusion matrix will be displayed within the main view.

The Confusion Matrix allows the admin to view the counts of the true positives and false negatives from the classification model.
